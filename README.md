# 🎨 JavaScript Drawing App

A simple drawing app built using JavaScript where users can draw on canvas.

## 🚀 Features
- Draw using mouse
- Change brush color
- Adjust brush size
- Clear canvas
- Save Drawing as image

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Canvas API

## 📂 Project Structure
├── index.html
├── style.css
├── script.js

This project was built while learning JavaScript and Canvas.
